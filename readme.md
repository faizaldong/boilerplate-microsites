# Start Micros!
1. git clone the project
2. npm install

# Create your own project
Run:
```sh
$ grunt create
```

# Remove your own project
Run:
```sh
$ grunt remove
```

# Duplicate your own project
Run:
```sh
$ grunt duplicate
```

# rename your own project
Run:
```sh
$ grunt rename
```

# Run your project
Run:
```sh
$ PROJECT=projectname grunt
```
